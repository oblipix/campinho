# 🎥 Filmes em Cartaz - Projeto React

#### Descrição do Projeto

Este projeto é uma aplicação React para exibir listas de filmes em cartaz, tendências e por gênero, utilizando a API do The Movie Database (TMDB). A interface permite navegar entre seções, visualizar detalhes dos filmes, assistir trailers diretamente e explorar os mais recentes lançamentos do cinema.

#### Funcionalidades

Filmes em Cartaz: Lista de filmes atualmente em exibição nos cinemas.
Tendências: Filmes mais populares do dia.
Filmes por Gênero: Categorias como Terror, Comédia, Drama, Ação, Aventura e mais.
Sistema de Carrossel: Navegação por slides para explorar filmes.
Modal com Trailer: Detalhes do filme e trailer exibidos em um modal.
Paginador Dinâmico: Carregamento de múltiplas páginas de resultados para filmes em cartaz.


#### Tecnologias Utilizadas

Frontend: React.js
Estilização: CSS personalizado
API: The Movie Database (TMDB)
